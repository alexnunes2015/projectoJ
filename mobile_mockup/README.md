# Mobile Mockup

Mockup Flutter de uma unica pagina para simular o envio de uma ocorrencia.

## O que existe

- Botao para simular fotografia
- Campo de categoria
- Campo de descricao
- Botao para simular localizacao
- Botao para enviar

Nada esta ligado a camera, GPS, backend ou base de dados. E apenas um mockup visual.

## Como gerar a app quando tiveres Flutter instalado

1. Entrar na pasta:

```bash
cd mobile_mockup
```

2. Gerar a estrutura nativa em falta:

```bash
flutter create .
```

3. Executar em dispositivo ou emulador:

```bash
flutter run
```

4. Gerar APK:

```bash
flutter build apk
```
