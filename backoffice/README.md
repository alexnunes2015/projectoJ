# Backoffice

Front-end React separado do backend.

## Estado atual

- App Vite + React.
- Login real contra o backend em `/api/auth/login`.
- Dashboard inicial com leitura de municípios, ocorrências, categorias e utilizadores.

## Como correr

```bash
cd backoffice
npm install
npm run dev
```

Abre `http://localhost:4173`.
